Las Mejores Flash-Cards del Mundo

HTML

Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet

CODIGO

Conjunto de instrucciones y datos a ser procesados automáticamente por una computadora

CSS

El concepto se utiliza en el ámbito de la informática para referirse a un lenguaje empleado en el diseño gráfico.

WEB

Conjunto de información que se encuentra en una dirección determinada de internet.

HTTPS

Protocolo Seguro de Transferencia de Hipertexto, empleada con frecuencia en el ámbito de la informática.

DOMINIO

Nombre fácil de recordar asociado a una dirección IP física de Internet.

FORMULARIO

Conjunto de campos de datos solicitados por un determinado programa

NEOCITES

Es un servicio de alojamiento web comercial para páginas estáticas

BLOG

Página web que generalmente de carácter personal, con una estructura cronológica que se actualiza regularmente

ARQUITECTURA

Estructuración del sistema que, idealmente, se crea en etapas tempranas del desarrollo

HEAD

Provee información general acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

RAM

Memoria principal de la computadora, donde residen programas y datos.

DISEÑO

Planificación de una solución de software

ETIQUETA

Palabra clave asignada a un dato almacenado en un repositorio

BACKEND

Es la parte de la app que el usuario final no puede ver

BACKUP

Copia y archivo de datos de la computadora de modo que se puede utilizar para restaurar la información original

INTERNET

Red informática de nivel mundial que utiliza la línea telefónica para transmitir la información.

BASE DE DATOS

Programa capaz de almacenar gran cantidad de datos, relacionados y estructurados

SERVIDOR

Ordenador que pone recursos a disposición a través de una red

FRONTEND

Parte del sitio web con la que el usuario interactúa.
